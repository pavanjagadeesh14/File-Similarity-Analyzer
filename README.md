# PlagiarismChecker


This project is a **file-based plagiarism checker** developed using **Java**, implementing **DSA**, **OOP principles**, and **SQL database integration**. It reads two `.txt` files, compares their content line by line, and calculates the plagiarism percentage using `HashSet` logic.

---

##  Features

-  Reads and cleans file content
-  Compares lines using efficient DSA logic
-  Detects similarity in textual content
-  Stores result in **MySQL** with JDBC
-  Clean modular OOP structure

---

## Tech Stack

-  Java (JDK 8+ or 21)
-  DSA (HashSet, List)
-  OOP (class separation)
-  MySQL + JDBC
-  Java standard libraries (I/O, SQL)
